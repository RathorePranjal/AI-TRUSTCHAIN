# AI Plagiarism Checker Implementation Summary

## Overview

This project implements an AI plagiarism checker that can detect AI-generated content in text. The system uses a combination of heuristic analysis and machine learning techniques to determine the likelihood that a given text was generated by an AI.

## Key Features

1. **Text Analysis**: Detects AI-generated content in text using heuristic analysis
2. **Multi-format Support**: Designed to support various file types (text, images, videos, code)
3. **Web Interface**: User-friendly frontend for content analysis
4. **API Service**: RESTful backend API for integration
5. **Blockchain Integration**: Certificate minting for content authenticity verification

## Implementation Details

### Backend (Node.js + Python)

The backend consists of:
- A Node.js Express server that handles API requests
- A Python script that performs AI text detection using heuristic analysis
- A mock detector as a fallback for demonstration purposes

#### AI Detection Logic

The AI detection uses several heuristics to determine if text is AI-generated:
- Word count analysis (longer texts are more likely to be AI-generated)
- Average word length (complex words may indicate AI generation)
- Punctuation density (less punctuation may indicate AI generation)
- Business jargon detection (words like "leverage", "paradigm", "synergy" are common in AI-generated text)

### Frontend (React)

The frontend provides a user interface for:
- Text input for analysis
- File upload for various file types
- Display of analysis results
- Certificate management

## API Endpoints

- `POST /api/analyze/text`: Analyze text for AI-generated content
- `POST /api/analyze/file`: Analyze uploaded files
- `POST /api/certificate/mint-onchain`: Mint authenticity certificates on blockchain
- `GET /api/certificate/list`: List all certificates

## Test Results

The system was tested with various text samples:

1. **Human-written text**: "This is a simple human-written sentence with normal punctuation and structure."
   - Result: Human-Likely (97% confidence)

2. **AI-generated text**: "Leveraging cutting-edge machine learning paradigms, this comprehensive analysis..."
   - Result: AI-Likely (53% confidence)

3. **Complex human-written text**: "Artificial intelligence is revolutionizing industries across the globe..."
   - Result: Human-Likely (79% confidence)

## Setup Instructions

1. Install Node.js dependencies:
   ```
   cd frontend
   npm install
   
   cd ../backend
   npm install
   ```

2. Install Python dependencies:
   ```
   cd backend
   pip install -r requirements.txt
   ```

3. Start the services:
   ```
   cd backend
   npm start
   
   cd ../frontend
   npm start
   ```

## Conclusion

This implementation provides a solid foundation for an AI plagiarism checker that can be extended to support multiple file types and more sophisticated detection algorithms. The current heuristic-based approach provides reasonable accuracy for demonstration purposes and can be replaced with more advanced ML models for production use.
